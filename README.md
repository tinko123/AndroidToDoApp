# ToDo приложение (Android, Kotlin)

Просто мобилно приложение за управление на задачи. Поддържа добавяне, преглеждане, редактиране, изтриване и отбелязване на задачите като изпълнени. Данните се съхраняват локално чрез Room база данни и се запазват след рестарт. Приложението съдържа три основни екрана: списък със задачи, екран за добавяне и екран за редакция. Реализирани са пълни CRUD операции и поддръжка на много редове описание и български език.

## Стартиране
1. Клонирайте репото.
2. Отворете проекта в Android Studio.
3. Стартирайте приложението през емулатор или устройство.
